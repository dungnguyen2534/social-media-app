## **Socius - Social Media App**

A modern **Next.js** social media platform mimicking Facebook/Instagram functionality. Built to showcase 
complex component architecture and seamless interactions in a responsive web environment.
- **Interactive Content System:**  Developed full CRUD functionality for posts utilizing Tiptap for composition and UploadThing for optimized media handling. Enhanced user engagement with a custom threaded comment system featuring Tenor GIF integration.
- **State Management & UX:**  Engineered a fluid user experience by implementing optimistic UI updates and streamlining data mutations via React Query and Server Actions. This ensures seamless visual feedback for CRUD operations and high-frequency interactions like liking and bookmarking.
- **Component Architecture & UI:** Constructed a highly accessible interface using Shadcn/ui and Tailwind CSS, achieving 97-100 Lighthouse scores and a flicker-free Dark/Light theme toggle.
- **Real-Time Messaging:** Integrated Stream Chat SDK to power a robust direct messaging system, fully customizing UI components for design consistency.
- **Authentication:** Secured user access using NextAuth.js with Google OAuth and passwordless Magic Links (Resend).
