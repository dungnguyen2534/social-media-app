## **Socius - Social Media App**

A modern **Next.js** social media platform replicating core Facebook and Instagram functionality. Built to demonstrate complex component architecture and seamless user interactions within a responsive web environment.

- **Authentication & Security:** Implemented robust authentication using **NextAuth.js**, featuring **Google OAuth** and **passwordless Email Magic Links** with **Resend** to ensure secure and reduce entry barriers for users.
- **Interactive Content System:** Built a comprehensive CRUD interface for user content. Users can customize profiles, utilize **Tiptap editor** to compose posts with image/video attachments, and engage via **threaded comments** featuring **Tenor GIF** integration.
- **State Management & UX:** Engineered a fluid user experience by implementing **optimistic UI updates** and streamlining data mutations via **React Query** and **Server Actions**. This ensures seamless visual feedback for CRUD operations and high-frequency interactions like liking and bookmarking.
- **Component Architecture & UI:** Constructed a highly responsive, accessible interface using **shadcn/ui** and **Tailwind CSS**, achieving **near-perfect Lighthouse scores (99-100)** across all categories. Implemented dynamic layouts, reusable component patterns, and a flicker-free Dark/Light theme toggle.
- **Real-Time Messaging:** Integrated **Stream Chat SDK** to power a robust direct messaging system, fully customizing chat UI components to ensure design consistency across the application.
- **Media Handling:** Developed a robust file upload interface supporting **drag-and-drop uploads** with image and video previews. Integrated **UploadThing** for efficient client-side validation and media rendering.
